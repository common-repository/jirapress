== Description ==

jiraPress is a simple wordpress plugin that takes jira issue-tracker number and converts them to hyperlinks.

== Installation ==

= Installation =
1. Download the zip file and extract the contents.
1. Upload the 'jirapress' folder to your plugins directory (wp-content/plugins/).
1. Activate the plugin through the 'plugins' page in WP.
